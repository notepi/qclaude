# Space Intel 股票池治理规则 v1.0

> 版本：2026-03-18  
> 适用范围：商业航天板块股票池配置与维护  
> 目标：确保 benchmark 质量，避免污染，提升结论可靠性

---

## 一、分层定义

### 1. anchor（锚定标的）
- **定义**：主要研究标的，所有分析围绕其展开
- **当前成员**：688333.SH 铂力特
- **计算规则**：不参与任何均值/排名计算，作为被分析对象
- **变更频率**：极低，除非研究主线切换

### 2. trading_core（交易核心层）
- **定义**：用于计算 relative_strength、排名、rolling 的基准股票池
- **当前成员**：600343.SH 航天动力、600879.SH 航天电子、600118.SH 中国卫星
- **计算规则**：
  - sector_avg_return = trading_core 成员当日平均涨跌幅
  - relative_strength = 铂力特涨跌幅 - sector_avg_return
  - 排名分母 = trading_core 成员数 + 1（含铂力特）
- **核心要求**：
  - 与铂力特交易联动性强（近20日相关系数 ≥ 0.60）
  - 商业航天主线相关，非周期/跨界污染
  - 样本量 3~5 只为宜，避免单只异动过度影响均值

### 3. research_core（研究核心层）
- **定义**：与铂力特业务逻辑直接相关的研究对标，不参与 benchmark 计算
- **当前成员**：600343.SH 航天动力、688433.SH 华曙高科、003009.SZ 中天火箭
- **计算规则**：
  - research_avg_return = research_core 成员当日平均涨跌幅
  - research_relative_strength = 铂力特涨跌幅 - research_avg_return
- **核心要求**：
  - 产业链位置相关（上游/下游/同赛道）
  - 研究逻辑可比（驱动因子、估值逻辑、政策敏感度）
  - 交易联动性非必须，但需持续观察

### 4. trading_candidates（交易候选池）
- **定义**：交易联动性强，待评估后可能升入 trading_core
- **当前成员**：601698.SH 中国卫通、688102.SH 斯瑞新材
- **观察重点**：
  - 近20日/近10日相关系数稳定性
  - 是否出现与商业航天主线脱钩的独立行情
  - 是否引入非航天因素（如材料价格波动、通信政策）

### 5. research_candidates（研究候选池）
- **定义**：业务逻辑相关，待评估后可能升入 research_core
- **当前成员**：（暂空，斯瑞新材已评估后保留为 trading_candidate）
- **评估维度**：
  - 产业链位置是否构成有效参照
  - 研究逻辑是否直接可比
  - 数据完整性

### 6. extended_watchlist（扩展观察池）
- **定义**：与商业航天有一定关联，但逻辑不纯或联动较弱，只观察不参与计算
- **当前成员**：600151.SH 航天工程、688062.SH 航天宏图、688568.SH 中科星图、002179.SZ 中航光电、603678.SH 火炬电子、000733.SZ 振华科技、603267.SH 鸿远电子、300762.SZ 上海瀚讯
- **用途**：
  - 监控板块整体情绪
  - 发现潜在候选股
  - 验证 trading_core 代表性

### 7. remove / 降级观察
- **定义**：已从核心层移除，保留观察记录供追溯
- **当前成员**：600151.SH 航天工程（原 trading_core，因主业为煤化工降级）
- **降级原因记录**：必须在 reason 字段明确说明

## 二、准入规则

### trading_core 准入标准

必须同时满足：

1. **相关性门槛**
   - 近20日相关系数 ≥ 0.60
   - 近10日相关系数 ≥ 0.50
   - 连续5个交易日无显著反向走势（单日偏离 > 5%）

2. **业务相关性**
   - 商业航天制造/发射/卫星主线相关
   - 非煤化工、非纯周期股、非跨界概念
   - 与铂力特共享至少一个核心驱动因子

3. **数据质量**
   - 历史数据 ≥ 20 个交易日
   - 无长期停牌、无流动性问题

4. **样本量控制**
   - trading_core 总数 ≤ 5 只
   - 新增需评估是否替换现有成员，而非简单扩池

### research_core 准入标准

满足其一即可：

1. **同赛道直接可比**
   - 与铂力特同属增材制造赛道（如华曙高科）
   - 估值逻辑、驱动因子高度一致

2. **产业链上下游**
   - 铂力特的直接客户（如中天火箭，火箭制造需求→零件订单）
   - 铂力特的直接供应商（如高端金属粉末厂商）

3. **研究验证价值**
   - 可作为铂力特定价/订单/产能的先行指标
   - 研究逻辑互补，能解释铂力特无法覆盖的行业维度

### candidate 观察标准

- **trading_candidate**：交易联动性初筛通过（全量相关系数 > 0.50），但需观察稳定性
- **research_candidate**：业务逻辑初筛通过，但需验证数据完整性和研究可比性

### extended_watchlist 用途

- 持仓结构、行业属性与商业航天有交叉
- 但交易联动性不足或研究逻辑不直接相关
- 用于板块情绪监控，不进入任何计算层

## 三、移出规则

### 强制移出（立即生效）

1. **主业不符**
   - 实际主营业务与商业航天无实质关联
   - 案例：600151.SH 航天工程（主业为煤化工工程）

2. **长期联动失效**
   - 连续20个交易日相关系数 < 0.30
   - 或出现系统性脱钩（如受独立政策/周期驱动）

3. **持续污染 benchmark**
   - 纳入后导致 relative_strength 系统性失真
   - 经回测验证，纳入前后结论方向频繁翻转

### 观察期移出（review_date 到期后评估）

1. **review_date 到期复审不通过**
   - 未达到准入标准的硬性门槛
   - 或出现新的负面因素（如业绩暴雷、行业政策变化）

2. **更优候选出现**
   - 新候选股在相关性/逻辑纯度上显著优于现有成员
   - 经评估后替换而非简单扩池

## 四、复审机制

### review_date 含义

- 下次必须重新评估该股票分层的日期
- 不是自动升级/降级，而是强制触发评估流程
- 评估结果：升入核心 / 保持候选 / 降级观察 / 移出

### 复审频率

| 分层 | 默认 review_date | 触发条件 |
|------|:---:|------|
| trading_core | 每季度（90天） | 相关系数连续下滑、出现独立行情 |
| research_core | 每季度（90天） | 研究逻辑变化、数据质量问题 |
| trading_candidate | 30天 | 相关性观察期 |
| research_candidate | 30天 | 研究价值验证期 |
| extended_watchlist | 90~180天 | 低优先级观察 |

### 复审指标

**交易层复审**：
- 近20日相关系数（门槛 0.60）
- 近10日相关系数（门槛 0.50）
- 近5日是否出现方向相反（单日偏离 > 5%）
- 近20日涨跌幅偏离度（与铂力特差异 > 10% 需说明）

**研究层复审**：
- 研究逻辑是否仍然成立（产业链关系变化）
- 数据完整性（缺失率 < 5%）
- 是否有新的研究价值验证（如订单公告、产能扩张）

## 五、当前实战规则固化

### 中国卫通（601698.SH）暂缓进 core 的原因

- **现状**：trading_candidate，active=false
- **数据**：已 fetch 37 条，近20日相关系数 +0.565，近10日 +0.222
- **问题**：近期走势严重分化，近10日基本失联
- **门槛**：需近20日 ≥ 0.65，近10日 ≥ 0.50，连续5日无反向
- **review_date**：2026-04-17

### 斯瑞新材（688102.SH）作为 trading_candidate 的原因

- **现状**：trading_candidate（2026-03-18 从 research_candidate 调整）
- **数据**：已 fetch 37 条，全量 +0.866，近20日 +0.821，近10日 +0.812
- **优势**：交易联动性极强，所有股票中最高
- **顾虑**：材料端属性（铜合金/难熔金属）与制造端主线不完全对齐
- **决策**：保留为 trading_candidate，不入 core，避免材料周期污染 benchmark
- **review_date**：2026-04-17

### 上海瀚讯（300762.SZ）保留在 extended_watchlist 的原因

- **现状**：extended_watchlist（2026-03-18 从 trading_candidate 调整）
- **数据**：已 fetch 37 条，全量 +0.815，近20日 +0.864，近10日 +0.894
- **优势**：交易联动性极强，近10日相关系数所有股票中最高
- **问题**：产业链位置过远（卫星通信应用端 vs 增材制造），研究逻辑不相关
- **决策**：不入任何核心层，保留观察，监控板块情绪即可
- **review_date**：2026-06-17（观察期更长）

### 航天工程（600151.SH）降级的原因

- **原分层**：trading_core（core_universe）
- **降级日期**：2026-03-17
- **原因**：主业为煤化工工程（甲醇/合成氨），与商业航天无实质关联
- **影响**：移出前 sector_avg 被系统性拉高，铂力特 relative_strength 被低估
- **现状**：extended_watchlist，benchmark_included=false，不参与任何计算
